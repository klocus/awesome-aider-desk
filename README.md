## 🔌 Extensions

This repository hosts community-contributed extensions for AiderDesk. Extensions are TypeScript-based plugins that can extend AiderDesk's functionality with new tools, commands, UI components, and custom workflows.

📖 **Full documentation**: See [extensions/README.md](extensions/) for installation instructions and contribution guidelines.

## Contributing

Contributions are welcome! Whether you've created an extension, improved documentation, or found a bug, we'd love to hear from you.

### Adding an Extension

To add your extension to this repository:

1. Fork this repository
2. Create a new directory in `extensions/` with your extension name
3. Add your extension files and a `README.md` with usage instructions
4. Submit a pull request
